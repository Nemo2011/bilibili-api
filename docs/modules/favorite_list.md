from bilibili_api import favorite_list

收藏夹操作。

## class FavoriteListContentOrder

**Extends:** enum.Enum

收藏夹列表内容排序方式枚举。

+ MTIME  : 最近收藏
+ VIEW   : 最多播放
+ PUBTIME: 最新投稿


## class FavoriteListType

**Extends:** enum.Enum

收藏夹类型枚举

+ VIDEO  : 视频收藏夹
+ ARTICLE: 专栏收藏夹
+ CHEESE : 课程收藏夹
---

## class SearchFavoriteListMode

**Extends:** enum.Enum

收藏夹搜索模式枚举

+ ONLY : 仅当前收藏夹
+ ALL  : 该用户所有收藏夹
